Projects

Disney Princess Quiz

Full Stack

Created for my Disney Princess obsessed daughter Princess Evelyn. This application was designed to provide an empowering journey for aspiring Princesses who must rid the kingdom of troublesome Princes. The frontend was created using Vue.js which sends HTTP requests to a custom made REST API. This API provides functionality to retrieve questions from a MongoDB database and also to insert new questions into that database.

Pints and Playgrounds

Full Stack

A website designed to allow users to locate their nearest pub that also has a playground for the kiddos. Once users have signed up, they can submit their own local pubs which are then approved by an admin before being sent to the database. This is a full stack application created using, EJS on the frontend and node.js with express.js on the backend. Pub and User data are stored in a MongoDB database.

  • JavaScript
  • EJS
  • Node.js
  • HTML
  • CSS
  • MongoDB

Super Speller

Frontend

Created to help my daughter practice her spelling. The user selects a difficulty and this dictates whether images of three, four or five letter words are shown. The user then has three lives to score as many points as possible by spelling the words correctly. The game also features a leaderboard to add a bit of competition!

  • JavaScript
  • Vue.js
  • HTML
  • CSS

Trackman

Full Stack

A dummy website designed to assist law enforcement agencies in tracking and locating stolen vehicles. Users are required to sign up before they are able to view a map with pins showing the location of stolen vehicles. Vehicles can be added as well as having their location and status updated by certain authorised users. An EJS frontend communicates with a Node.js backend and a MySQL database.

  • JavaScript
  • EJS
  • Node.js
  • HTML
  • CSS
  • MySQL
